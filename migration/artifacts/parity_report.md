## Performance

**Thread mode**: warm p50/p95 `multi-thread`; cold `multi-thread`. Built-in speed trials: `3` using `median-of-trial-percentiles`.
Single-thread mode caps thread pools only; SIMD/ILP remain enabled within each CPU core.
Timing policy: Canonical parity speed timings run 3 serial trials per API/lane and gate on the median of per-trial p50/p95 estimates.
