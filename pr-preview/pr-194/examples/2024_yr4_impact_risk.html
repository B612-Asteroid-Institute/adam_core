So you Want to Calculate an Impact Risk¶

Here we walk you through the steps necesssary to calculate the impact risk for a specified orbit, in this case 2024 YR4. We’re going to take a few steps:

  1. Fetch the state vector for 2024 YR4 from JPL Horizons

  2. Run calculate impacts, it will create and propagate samples through time and check for collisions.

  3. Summarize the results into an impact risk.

Before we can start, you will need to install a couple packages.

pip install adam-core[plots] adam-assist

1# Fetch the state vector from Small Bodies Database (SBDB)
2from adam_core.orbits.query import query_sbdb
3
4orbit = query_sbdb(["2024 YR4"])
5
6print(orbit.to_dataframe())

1# We need to know how many days out to run the propagation,
2# relative to when the orbit state vector is defined.
3# Let's go 30 days past the possible 2032 impact of 2024 YR4
4from adam_core.time import Timestamp
5approx_impact_date = Timestamp.from_iso8601(["2032-12-22"], scale="tdb")
6thirty_days_after_impact = approx_impact_date.add_days(30)
7days_until_thirty_days_after_impact, _ = thirty_days_after_impact.difference(orbit.coordinates.time)
8print(days_until_thirty_days_after_impact)

 1# Now we initialize our propagator, and pass it to our calculate_impacts function
 2from adam_core.dynamics.impacts import calculate_impacts, CollisionConditions
 3from adam_core.coordinates import Origin
 4from adam_assist import ASSISTPropagator
 5
 6propagator = ASSISTPropagator()
 7
 8# Define the collision conditions including a potential impact on the Moon
 9conditions = CollisionConditions.from_kwargs(
10    condition_id=["Earth", "Moon"],
11    collision_object=Origin.from_kwargs(code=["EARTH", "MOON"]),
12    collision_distance=[6420, 1740],
13    stopping_condition=[True, True],
14)
15
16variants, impacts = calculate_impacts(
17    orbit,
18    days_until_thirty_days_after_impact[0].as_py(),
19    propagator,
20    num_samples=10000,
21    processes=60, # Multiprocessing speeds things up if you have the CPUs
22    conditions=conditions,
23)
24
25print(variants, impacts)

1# Now we can summarize the returns, a simple ratio in our case
2# with only 1 orbit considered
3from adam_core.dynamics.impacts import calculate_impact_probabilities
4ip = calculate_impact_probabilities(variants, impacts, conditions=conditions)
5ip.to_dataframe()

We’ve recently added some visualization functionality to visualize both the risk corridor and the impact events. Let’s first look at the impact corridor.

1from adam_core.dynamics.plots import plot_risk_corridor
2
3fig = plot_risk_corridor(impacts, title="Risk Corridor for 2024 YR4")
4fig.show()

 1from adam_core.dynamics.plots import generate_impact_visualization_data
 2
 3propagation_times, propagated_best_fit_orbit, propagated_variants = generate_impact_visualization_data(
 4    orbit,
 5    variants,
 6    impacts,
 7    propagator,
 8    time_step=5,
 9    time_range=60,
10    max_processes=None
11)

 1from adam_core.dynamics.plots import plot_impact_simulation
 2
 3fig = plot_impact_simulation(
 4    propagation_times,
 5    propagated_best_fit_orbit, 
 6    propagated_variants, 
 7    impacts, 
 8    title="2024 YR4 Impact Simulation (Simulated on 2025-03-14)",
 9    sample_impactors=None,
10    sample_non_impactors=0.1
11)
12fig.write_html("2024YR4_impact_simulation.html")

And there you have it.
