Welcome to adam_core
=================

``adam_core`` is a comprehensive Python library for planetary science, developed by the B612 Asteroid Institute. It provides tools for working with orbital mechanics, coordinate systems, observations, and more.

Key Features
-----------

* Robust coordinate system transformations
* Orbital propagation and ephemeris generation
* Observation handling and processing
* Time system conversions
* SPICE kernel integration
* High-performance computations
